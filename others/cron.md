# Cron

### Format of cron

```text
minute hour day_of_month month day_of_week  command_to_execute

# Possible values: 
    *	any value
    ,	value list separator
    -	range of values
    /	step values
    @yearly	(non-standard)
    @annually	(non-standard)
    @monthly	(non-standard)
    @weekly	(non-standard)
    @daily	(non-standard)
    @hourly	(non-standard)
    @reboot	(non-standard)
```
